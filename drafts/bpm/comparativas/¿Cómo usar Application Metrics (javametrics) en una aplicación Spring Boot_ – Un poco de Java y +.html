¿Cómo usar Application Metrics (javametrics) en una aplicación Spring Boot?

Date: 13 julio 2018Author: LuisMi Gracia 0 Comentarios

Application Metrics for Java es una librería opensource con licencia Apache desarrollado por DeveloperWorks que instrumenta el Runtime de Javas para ofrecernos información de monitorización de forma visual en un dashboard.
