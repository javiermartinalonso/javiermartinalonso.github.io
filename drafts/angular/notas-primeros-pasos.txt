intenta esto si estás usando Linux

	sudo killall ng
	
si dice lo contrario, dice que el puerto todavía está en uso cuando se reinicia, intente seguir para detener el progreso en el puerto 8080:

sudo kill $(sudo lsof -t -i:8080)



intenta esto si estás usando Windows

En cmd.exe intente encontrar el PID del puerto 8080: 

	netstat -ano | findstr :8080	

Luego elimine el proceso que tiene la Propiedad "LISTENING" con este comando (el número en el comando puede diferir): 

	taskkill /PID 4480 /F
